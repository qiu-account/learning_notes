2.至少一项工作的详细过程说明；
  毕设的最后一个评估指标--对正常任务的干扰波动：
  任务干扰波动响应用于评估安全干预对非目标任务的影响程度，即该方向是否能够在抑制危险输出的同时，尽量保持模型在合法任务中的正常表现。为全面刻画干预带来的副作用，本文从任务性能、内部表征以及输出行为三个层面进行多维度评估。
为统一度量干预影响，本文对各项指标均计算干预前后变化量，记为：ΔM=Mafter−Mbefore。具体包括以下四个子维度：
（1）知识问答任务性能：使用MMLU心理学子集，计算干预前后模型在选择题上的准确率变化，用于评估干预对知识推理能力的影响。
（2）内部表征稳定性：计算干预前后模型在指定层的隐藏状态相似度，本文计算干预前后隐藏状态向量之间的相似度。具体地，对于同一输入，分别提取干预前后的表示向量，并采用余弦相似度进行度量并分析其变化情况，以评估干预对模型内部表示结构的扰动程度。
（3）日常对话任务表现：构造涵盖常识问答、日常对话、写作、推理及建议等多类问题的数据集，从输出质量角度评估干预影响。具体包括文本相似度（衡量语义保持程度）、拒绝率（衡量是否出现不必要的拒绝）以及平均输出长度（反映生成行为变化）。
（4）敏感问题边界鲁棒性：使用XSTest数据集，评估模型在语义安全但包含敏感词的边界样本上的拒绝率变化，用于检验干预是否引入过度防御现象。

4.实践总结（不少于300字）
这次读书实践周，我收获还是颇丰的。
先说毕设这边。目前实验部分已经基本跑完，初稿也整理得差不多了，接下来就是认真完善、打磨定稿。回顾整个大四做毕设的过程，说实话，确实挺磨人的，从一开始的迷茫、不知道从哪下手，到中间反复调实验、改指标，再到现在的初稿成型，每一步都得花不少时间和精力。但话说回来，看到自己的东西一点一点做出来，最后有个像样的结果，那种开心也是实实在在的。做科研大概就是这样吧，投入和收获是成正比的，虽然过程有时候挺煎熬，但回头看还是觉得值。
除了毕设本身，这次实践周还有一个意外的收获，就是借机让自己慢下来了。大四这一年事情特别多，毕设、工作、各种材料，经常感觉被推着走。但这次读《被讨厌的勇气》，加上实践周相对集中地做一件事情，反而让我意识到：很多事情急也没用，与其焦虑地赶进度，不如稳下来、静下来，找到自己最好的状态再去推进。这个状态对了，效率反而更高，思路也更清楚。
总之，这10天既让我在毕设上往前推了一大步，也让我在心态上有了点变化。接下来的任务就是沉住气，把毕设定稿打磨好，顺利走完大学最后一程~
